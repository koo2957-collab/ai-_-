# 📱 모바일 업무 네비게이터

외부에서 체크리스트를 확인하고 AI 길잡이를 받는 모바일 업무 관리 앱

## ✨ 주요 기능

- 📦 **업무 가져오기**: 데스크탑에서 작업한 업무 패키지 가져오기
- ⚙️ **API 설정**: Gemini, ChatGPT, Claude API 키 설정
- 📋 **체크리스트 보기**: 업무별 체크리스트 확인 및 완료 상태 관리
- 💡 **AI 길잡이**: 체크리스트 항목별 상세 가이드 제공

## 🚀 설치 방법

### GitHub Pages 배포 (권장)

1. 이 저장소를 클론하거나 다운로드
2. GitHub에 업로드
3. Settings → Pages에서 GitHub Pages 활성화
4. `https://사용자명.github.io/저장소명/` 접속
5. 브라우저에서 "홈 화면에 추가" 선택

### 직접 사용

1. `모바일_업무_네비게이터.html` 파일 다운로드
2. 모바일 브라우저로 열기
3. "홈 화면에 추가" 선택

## 📱 사용 방법

1. **API 설정**: 상단 [⚙️ API] 버튼으로 API 키 설정
2. **업무 가져오기**: 상단 [📦 가져오기] 버튼으로 업무 패키지 가져오기
3. **체크리스트 확인**: 업무 카드를 탭하여 체크리스트 보기
4. **AI 길잡이**: 체크리스트 항목의 [💡 AI] 버튼으로 가이드 확인

## 🔒 보안

- ✅ API 키는 로컬에만 저장 (AES-GCM 암호화)
- ✅ 업무 데이터는 IndexedDB에 저장
- ✅ 깃허브에 올려도 API 키 노출 없음
- ✅ 각 사용자 데이터는 해당 기기에만 존재
